# Terraforming

For you to host terraform, you will need to create a file called terraform.tfvars. The structure should be as follows, with every field filled with your own details.

ACCESS_KEY=
SECRET_ACCESS_KEY=
AWS_REGION=
DB_HOST=
DB_NAME=
DB_PASSWORD=
DB_PORT=
DB_SCHEMA=
DB_USER=
PIPELINE_URI=
TRANSLOCATION_URI=
DASHBOARD_URI=
